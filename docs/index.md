# *Getting Started*

## Install JCalc

### Install by pip
    pip install jcalc

### Dependencies
* [GROMACS](http://www.gromacs.org/)
* [Biopython](https://biopython.org/)
* [Obabel](https://open-babel.readthedocs.io/en/latest/)

### Install by Docker
    docker pull jlmeirelles/jcalc:latest
